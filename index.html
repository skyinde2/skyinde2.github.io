Rendering

Kart Decal

  • User customizable car decals.

Optimized Blob Shadow

  • Works like a projector without flickering on uneven surfaces. The mesh for rendering is dynamically generated to reduce overdraw.

Capsule Shadow

  • Simulates the effect of spotlight shadows within a stadium for enhanced performance and visual representation.

Crowd Rendering

  • Bakes animation data into texture array and makes skinning calculation in shader.

Reflection Effect

  • Reflection objects are rendered in a separate pass and then sampled with distortion based on normals and ground height.

Post Processing

  • Includes some mobile-optimized post effects such as depth of field, radial blur, bloom, and LUT, filling in for missing features in earlier versions of Unity.

Instance rendering
